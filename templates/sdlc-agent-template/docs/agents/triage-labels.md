# Triage Labels

Etiquetas canonicas para el repositorio:

- `needs-triage`
- `needs-info`
- `ready-for-agent`
- `ready-for-human`
- `wontfix`

## Uso esperado

- `needs-triage`: requiere evaluacion inicial
- `needs-info`: falta contexto o respuesta del autor
- `ready-for-agent`: puede ejecutarse de forma autonoma con contexto suficiente y bloque `validation` resuelto
- `ready-for-human`: requiere implementacion o decision humana principal
- `wontfix`: no se accionara
